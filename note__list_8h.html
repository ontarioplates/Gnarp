Insert a given note in proper order into a NoteList.

If the Note is not a duplicate, it is inserted into the NoteList in proper pitch order and at the end of the trigger order. If the Note is a duplicate, its velocity is updated and moved to the end of the trigger order. If the #note_bank is filled, no new Note will be inserted.

Parameters:
note_listpointer to the NoteList to use
pitch8-bit pitch value of the new Note
velocity8-bit velocity value of the new Note
Returns:
1 if the new Note was inserted (including duplicates)
0 otherwise

Remove a given note from a NoteList.

This function searches for the NoteList for a Note with the given pitch. If the Note is found, its note_bank position is freed and the surrounding Notes' pointers are adjusted.

Parameters:
note_listpointer to the NoteList to use
pitch8-bit pitch value to search for
Returns:
1 if the pitch was found and removed
0 otherwise
