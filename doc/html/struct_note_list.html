Detailed Description

This structure is a linked list of incoming MIDI notes.

This structure is an organized history of all the notes currently being played at the MIDI input jack.
The list can be traversed forwards or backwards in order of pitch or the order which the notes were triggered.
MAX_LIST_NOTES determines the maximum number of notes that can be stored in the list. Once this maximum is reached, newly inserted notes will not be tracked.
